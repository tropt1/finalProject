"Final Project" is a game developed in Unity, offering a comprehensive and immersive experience. The game leverages Unity's robust engine to deliver stunning graphics and engaging gameplay. ## Installation

Clone the repository

```git clone https://github.com/tropt1/finalProject.git```

### Prerequisites

Unity: Ensure you have Unity Hub installed along with the latest LTS version of Unity.

System Requirements: Make sure your system meets the necessary requirements to run a Unity game.

### Steps

1. Download the Game Files: Obtain the game files from the provided link or repository.
2. Extract the Files: Unzip the downloaded files to your desired location.
3. Run the Game: Navigate to the extracted folder and run the executable file finalProject.exe.
